HTML Basic Tags

HTML stands for HyperText Markup Language. It is used to create web pages.

Some basic HTML tags:

  • <h1> - Heading Tag
  • <p> - Paragraph Tag
  • <b> - Bold Tag
  • <i> - Italic Tag
  • <a> - Anchor Tag
