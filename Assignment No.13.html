Assignment No.13: Student Application using XSLT

Name: Suraj Govind Munde
Roll No: 18

This assignment demonstrates using XML and XSLT to transform student data into HTML with conditional remarks.

  • XML data: students.xml
  • XSL stylesheet: students.xsl
  • How to run: Open students.xml in a browser, which will apply students.xsl automatically.

If the browser does not render the result, open a local server in this folder and open http://localhost:8000/students.xml.
